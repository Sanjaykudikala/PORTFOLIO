Project 01 · Jan 2026 – Feb 2026
AI-Powered Presentation Generator
The Problem

Creating polished presentations manually is time-consuming. This project automates the entire process — from topic to styled slides — using LLMs, parallel processing, and a custom design system.

  • Built a LangGraph parallel worker architecture to generate slide content concurrently, slashing latency
  • Designed a Pydantic-validated planner module to define themes, layouts, and content blueprints
  • Integrated a custom design system (colors, typography, layouts) with python-pptx for dynamic styling

Project 02 · Nov 2025 – Dec 2025
Fake News Detection System
The Problem

Misinformation spreads fast. This system fine-tunes a DistilBERT transformer on a large-scale dataset to classify news articles as real or fake, with a fully interactive web app for inference.

  • Fine-tuned DistilBERT with early stopping, cosine LR scheduling, and mixed-precision (~7,500 steps)
  • Built Streamlit app for single-article and batch inference with Plotly confidence visualizations
  • Modular inference pipeline with tokenizer-based preprocessing — no external API dependency at runtime
Python DistilBERT HuggingFace Streamlit Plotly
⟢ GitHub
Project 03 · May 2025 – Jun 2025
Intelligent YouTube Query Bot
The Problem

YouTube videos contain hours of information that's hard to search. This RAG-powered chatbot extracts and indexes transcripts so users can ask questions and get accurate, context-aware answers instantly.

  • Implemented FAISS vector search indexing on video transcript embeddings for fast retrieval
  • Used Retrieval-Augmented Generation (RAG) pipeline with LangChain for accurate, grounded responses
  • Deployed publicly on Gradio Spaces for easy access and live demo
Python LangChain FAISS Gradio RAG
⟢ GitHub
Project 04 · Mar 2025 – Apr 2025
Content-Based Film Recommendation Engine
The Problem

Finding movies you'll love is hit or miss. This recommendation engine analyzes genre, cast, and plot metadata from 5,000 movies to suggest personalized picks using cosine similarity and NLP feature extraction.

  • Processed movie metadata with TF-IDF and Count Vectorizer for rich feature representation
  • Applied cosine similarity to compute movie-to-movie similarity scores at scale
  • Interactive Streamlit web UI for real-time personalized recommendations
Python TensorFlow Scikit-learn TF-IDF Streamlit
